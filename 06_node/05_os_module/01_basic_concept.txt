----------------------------------------------------------
1. WHAT IS OS MODULE
----------------------------------------------------------

The os module is a built-in Node.js module used to get information about the operating system.

Explanation:
It provides system-level information such as CPU details, memory usage, system architecture, platform, uptime, and more.

Developers use this module to monitor system resources or optimize applications based on the system environment.


----------------------------------------------------------
2. IMPORTING OS MODULE
----------------------------------------------------------

Since os is a core module, it does not require installation.

Example

const os = require("os")


----------------------------------------------------------
3. os.platform()
----------------------------------------------------------

Definition:
Returns the operating system platform.

Example

os.platform()

Possible Output

win32   → Windows
linux   → Linux
darwin  → macOS


----------------------------------------------------------
4. os.arch()
----------------------------------------------------------

Definition:
Returns the CPU architecture of the operating system.

Example

os.arch()

Output Example

x64


----------------------------------------------------------
5. os.cpus()
----------------------------------------------------------

Definition:
Returns information about the CPU cores.

Example

os.cpus()

Output Example

[
 { model: 'Intel...', speed: 2400 },
 { model: 'Intel...', speed: 2400 }
]

This tells how many CPU cores exist.


----------------------------------------------------------
6. os.totalmem()
----------------------------------------------------------

Definition:
Returns the total system memory in bytes.

Example

os.totalmem()


----------------------------------------------------------
7. os.freemem()
----------------------------------------------------------

Definition:
Returns the available free memory in bytes.

Example

os.freemem()


----------------------------------------------------------
8. os.hostname()
----------------------------------------------------------

Definition:
Returns the hostname of the operating system.

Example

os.hostname()

Example Output

DESKTOP-ABC123


----------------------------------------------------------
9. os.homedir()
----------------------------------------------------------

Definition:
Returns the home directory of the current user.

Example

os.homedir()

Example Output

C:\Users\vikash


----------------------------------------------------------
10. os.uptime()
----------------------------------------------------------

Definition:
Returns the system uptime in seconds.

Example

os.uptime()


----------------------------------------------------------
11. os.userInfo()
----------------------------------------------------------

Definition:
Returns information about the current logged-in user.

Example

os.userInfo()

Output Example

{
 username: "vikash",
 homedir: "C:\\Users\\vikash"
}


----------------------------------------------------------
12. os.networkInterfaces()
----------------------------------------------------------

Definition:
Returns network interface information.

Example

os.networkInterfaces()

Shows IP addresses and network details.


----------------------------------------------------------
13. os.tmpdir()
----------------------------------------------------------

Definition:
Returns the default temporary directory.

Example

os.tmpdir()


----------------------------------------------------------
14. COMMON INTERVIEW QUESTIONS
----------------------------------------------------------

Q1: What is the Node.js OS module?

Answer:
The OS module is a built-in Node.js module that provides information about the operating system such as CPU, memory, platform, and system uptime.


Q2: How do you check CPU information in Node.js?

Answer:
Using os.cpus()


Q3: How do you check total memory?

Answer:
Using os.totalmem()


Q4: How do you check the platform of the system?

Answer:
Using os.platform()
