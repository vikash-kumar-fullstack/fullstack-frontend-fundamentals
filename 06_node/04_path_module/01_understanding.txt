---------------------------------------------------------
1. WHAT IS PATH MODULE
----------------------------------------------------------

The path module is a built-in Node.js module used to work with file and directory paths.

Explanation:
Different operating systems use different path formats. Windows uses "\" while Linux and macOS use "/". The path module helps create and manipulate file paths in a way that works across all operating systems.

Example Windows Path

D:\nodejs\html\js\app.js

Example Linux Path

/nodejs/html/js/app.js


----------------------------------------------------------
2. IMPORTING PATH MODULE
----------------------------------------------------------

Since path is a core module, no installation is required.

Example

const path = require("path")


----------------------------------------------------------
3. STRUCTURE OF A FILE PATH
----------------------------------------------------------

Example path

D:\nodejs\html\js\app.js

Path Components

root : D:\
dir  : D:\nodejs\html\js
base : app.js
name : app
ext  : .js


----------------------------------------------------------
4. path.basename()
----------------------------------------------------------

Definition:
Returns the last portion of a path (the file name).

Syntax

path.basename(path, ext)

Example

const path = require("path")

const file = path.basename("D:/nodejs/html/js/app.js")

console.log(file)

Output

app.js

Removing extension

path.basename("app.js", ".js")

Output

app


----------------------------------------------------------
5. path.dirname()
----------------------------------------------------------

Definition:
Returns the directory name of a path.

Example

path.dirname("D:/nodejs/html/js/app.js")

Output

D:/nodejs/html/js


----------------------------------------------------------
6. path.extname()
----------------------------------------------------------

Definition:
Returns the file extension.

Example

path.extname("app.js")

Output

.js


----------------------------------------------------------
7. path.parse()
----------------------------------------------------------

Definition:
Returns an object containing all parts of a path.

Example

const result = path.parse("D:/nodejs/html/js/app.js")

console.log(result)

Output

{
 root: 'D:/',
 dir: 'D:/nodejs/html/js',
 base: 'app.js',
 ext: '.js',
 name: 'app'
}


----------------------------------------------------------
8. path.format()
----------------------------------------------------------

Definition:
Creates a path string from a path object.

Example

const obj = {
 dir: "D:/nodejs/html/js",
 base: "app.js"
}

console.log(path.format(obj))

Output

D:/nodejs/html/js/app.js


----------------------------------------------------------
9. path.isAbsolute()
----------------------------------------------------------

Definition:
Checks if a path is absolute.

Absolute path means the complete path from the root.

Example

path.isAbsolute("/users/app.js")

Output

true


Example

path.isAbsolute("app.js")

Output

false


----------------------------------------------------------
10. path.join()
----------------------------------------------------------

Definition:
Joins multiple path segments into a single path.

Example

path.join("users","nodejs","app.js")

Output (Linux/Mac)

users/nodejs/app.js

Output (Windows)

users\nodejs\app.js


----------------------------------------------------------
11. path.normalize()
----------------------------------------------------------

Definition:
Normalizes a path by resolving ".." and "." segments.

Example

path.normalize("/users/nodejs/../app.js")

Output

/users/app.js


----------------------------------------------------------
12. path.relative()
----------------------------------------------------------

Definition:
Returns the relative path from one location to another.

Example

path.relative("/data/test","/data/app")

Output

../app


----------------------------------------------------------
13. path.resolve()
----------------------------------------------------------

Definition:
Resolves a sequence of paths into an absolute path.

Example

path.resolve("app.js")

Output

C:\project\app.js


Example

path.resolve("folder","app.js")

Output

C:\project\folder\app.js
