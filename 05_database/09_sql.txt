1. WHAT IS SQL?
SQL (Structured Query Language) is used to interact with relational databases.

Explanation:
SQL allows us to create, read, update, and delete data from databases. It is the backbone of backend communication with RDBMS.

Example:
SELECT * FROM users;


------------------------------------------------------------


2. CLASSIFICATION OF SQL COMMANDS

1. DDL (Data Definition Language)
2. DML (Data Manipulation Language)
3. DQL (Data Query Language)
4. DCL (Data Control Language)
5. TCL (Transaction Control Language)


------------------------------------------------------------

DDL COMMANDS:-

3. CREATE TABLE
Used to create a new table.

Syntax:
CREATE TABLE users (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
);

Explanation:
Defines structure of table.


------------------------------------------------------------


4. ALTER TABLE
Used to modify an existing table.

Examples:

Add column:
ALTER TABLE users ADD email VARCHAR(100);

Modify column:
ALTER TABLE users MODIFY age INT;

Drop column:
ALTER TABLE users DROP COLUMN email;


------------------------------------------------------------


5. DROP TABLE
Used to delete a table permanently.

DROP TABLE users;


------------------------------------------------------------


6. TRUNCATE TABLE
Removes all data but keeps table structure.

TRUNCATE TABLE users;

Difference:
DELETE → removes row by row
TRUNCATE → removes all instantly


------------------------------------------------------------


DML COMMANDS:-

7. INSERT
Used to insert data.

INSERT INTO users (id, name, age) VALUES (1, 'Vikash', 22);


------------------------------------------------------------


8. UPDATE
Used to modify existing data.

UPDATE users
SET age = 23
WHERE id = 1;


------------------------------------------------------------


9. DELETE
Used to delete records.

DELETE FROM users
WHERE id = 1;


------------------------------------------------------------


DQL COMMANDS:-

10. SELECT
Used to retrieve data.

SELECT * FROM users;

SELECT name, age FROM users;


------------------------------------------------------------


11. WHERE CLAUSE
Used to filter data.

SELECT * FROM users
WHERE age > 18;


------------------------------------------------------------


12. OPERATORS
Comparison:
=, >, <, >=, <=, !=

Logical:
AND, OR, NOT

Example:
SELECT * FROM users
WHERE age > 18 AND name = 'Vikash';


------------------------------------------------------------


13. ORDER BY
Used to sort data.

SELECT * FROM users
ORDER BY age ASC;

DESC → descending


------------------------------------------------------------


14. LIMIT
Used to restrict number of rows.

SELECT * FROM users
LIMIT 5;


------------------------------------------------------------


15. DISTINCT
Removes duplicate values.

SELECT DISTINCT name FROM users;


------------------------------------------------------------


AGGREGATE FUNCTIONS:-

16. COUNT
gives the total numbers of rows

SELECT COUNT(*) FROM users;

---------------------------------
17. SUM
sum the values of one column

SELECT SUM(age) FROM users;
-------------------------------

18. AVG
SELECT AVG(age) FROM users;

19. MIN / MAX
gives max and min values in the particular column

SELECT MIN(age), MAX(age) FROM users;


------------------------------------------------------------


20. GROUP BY
Groups data.

SELECT age, COUNT(*)
FROM users
GROUP BY age;


------------------------------------------------------------


21. HAVING
Used with GROUP BY.

SELECT age, COUNT(*)
FROM users
GROUP BY age
HAVING COUNT(*) > 1;
